# 🌱 NayePankh Foundation Volunteer Management System

##  About NayePankh Foundation

NayePankh Foundation is a student-led NGO working towards **education, youth empowerment, skill development, social awareness, and community impact initiatives across India**.

it is a non-governmental organization (NGO) working towards social welfare, education, and community development.

The organization provides opportunities for students and volunteers to contribute to meaningful social causes.

##  Project Description

This project is a **Python-based Volunteer Management System** developed for NayePankh Foundation. It helps manage volunteer records in a simple and efficient way using a desktop application built with **Tkinter (Python GUI framework)** and file handling.

The system allows users to perform basic operations like adding, searching, deleting volunteers, and generating reports.

---

##  Features

-  Add Volunteer Details  
-  Search Volunteer by Name  
-  Delete Volunteer Record  
-  Generates Report   
-  Store Data using CSV File  
-  Simple and User-Friendly GUI 

---

##  Technologies Used

- Python 
- Tkinter (GUI Development)  
- CSV File Handling  
- File Input/Output (I/O)

---

##  Project Structure
- *main.py* → Contains full Tkinter GUI application 
- *volunteers.csv* → Stores all volunteer records 
- *report.txt* → Automatically generated summary report
- *README.md* → Project description and documentation for GitHub
  
---
  
 ##  How to Run the Project

1. Install Python (if not installed)  
   Download from: https://www.python.org/

2. Download or clone this repository

3. Open the project folder

4. Open `main.py` in any Python IDE  
   (VS Code / PyCharm / IDLE)

5. Click the **Run (▶) button** to execute the program

6. The Tkinter GUI window will open and the application will start

---

##  Output

- A desktop GUI window will open
- You can add, search, delete volunteers and generate report
- Data will be stored in `volunteers.csv`
- Report will be generated in `report.txt`
